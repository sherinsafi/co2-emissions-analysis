# 🌍 Global CO2 Emissions Analysis

## 📌 Project Overview
Analysis of global CO2 emissions from 1900-2024 
using Python and Power BI.

## 📁 Project Structure
- 📓 CO2_Emissions_Analysis.ipynb — Main notebook
- 📊 owid-co2-data.csv — Source dataset
- 📊 cleaned_co2_dataset.csv — Cleaned dataset
- 📄 Global_CO2_Emissions_Analysis_Report.pdf — Final report
- 🖼️ Global_CO2_Emission_Dashboard.pdf — Power BI Dashboard

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI
- Google Colab

## 📊 Dashboard
![Dashboard](Global_CO2_Emission_Dashboard.pdf)

## 💡 Key Insights
1. Coal accounts for 42.73% of all CO2 emissions
2. China is the largest emitter in 2024
3. Global CO2 has grown exponentially since 1950
4. 2020 COVID-19 caused largest single year drop
5. Wealthier nations emit more CO2 per person

## 🎯 Recommendations
1. Phase out coal with renewable energy
2. Target top 3 emitters — China, USA, India
3. Transfer clean technology to developing nations
4. Learn from COVID dip — promote remote work
